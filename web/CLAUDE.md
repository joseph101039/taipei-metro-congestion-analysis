# Metro Web

視覺化分析台北捷運壅塞狀況與分析洞見，提出改善方向的系統。

## Project Overview

This system provides:
- Visual analysis of Taipei Metro (MRT) congestion conditions
- Data-driven analytical insights
- Improvement direction recommendations based on congestion patterns

## Tech Stack

- **Framework**: React 19 + Vite
- **Dev server**: `npm run dev` (runs on port 3000)
- **Build**: `npm run build`

## Project Structure

```
src/
  App.jsx       # Main application component
  main.jsx      # Entry point
index.html      # HTML shell
vite.config.js  # Vite configuration
```
